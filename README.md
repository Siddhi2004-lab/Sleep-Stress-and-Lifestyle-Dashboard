# Sleep-Stress-and-Lifestyle-Dashboard
1) Project Title/ Headline:
😴Sleep, 🫀Health & 🧘‍♀️Lifestyle Analysis Dashboard.

2) Short Description/ Purpose:
This project presents an interactive dashboard that analyzes sleep patterns, stress levels and lifestyle habits using Power BI. The dashboard helps visualize relationships between sleep duration, physical activity, stress levels, and health indicators, providing insights into how lifestyle factors impact overall well-being.

3) Tech Stack:
The dashboard was built using the following tools and technologies:
📊 Power BI – Data visualization and dashboard creation.
🔄 Power Query – Data cleaning and transformation.
🧮 DAX (Data Analysis Expressions) – Creating calculated measures and KPIs.
📁 CSV Dataset – Data source used for analysis.

4) Data Source:
The dataset used in this project contains sleep and lifestyle-related information of individuals, including variables such as:
👤 Gender,🎂 Age,💼 Occupation,😴 Sleep Duration,⭐ Quality of Sleep,🧠 Stress Level,🏃 Physical Activity Level,❤️ Heart Rate,👣 Daily Steps,⚖ BMI Category,🩺 Blood Pressure,🛌 Sleep Disorder
This dataset helps analyze how lifestyle habits affect sleep quality and stress levels.

5) Features/ Highlights:
Key highlights of the dashboard include:
😴 Sleep Pattern Analysis – Visualizes sleep duration and sleep quality trends.
🧠 Stress Level Insights – Shows how stress levels vary across occupations and demographics.
🏃 Lifestyle Impact Analysis – Examines the relationship between physical activity, daily steps, and stress levels.
❤️ Health Indicators – Displays metrics such as heart rate and BMI categories.
🛌 Sleep Disorder Distribution – Identifies the prevalence of sleep disorders among individuals.
🎛 Interactive Filters – Allows users to filter data by gender, occupation, age, and BMI category.
📊 Data Visualization – Uses charts, KPIs, and graphs for clear and interactive insights.
